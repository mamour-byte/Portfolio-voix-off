Ajout des enregistrements publies

1. Placez vos fichiers audio dans ce dossier.
2. Ouvrez js/script.js.
3. Completez le tableau featuredRecordings avec :
   - title
   - category
   - language
   - description
   - src

Exemple :

{
  title: "Spot beaute premium",
  category: "Publicite",
  language: "Francais",
  description: "Voix chaleureuse et sophistiquee pour une campagne lifestyle.",
  src: "assets/audio/spot-beaute-premium.mp3"
}

Formats conseilles :
- MP3 pour la compatibilite
- WAV pour l'archivage ou la qualite studio
